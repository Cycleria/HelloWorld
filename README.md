# HelloWorld
HelloWorld in different program language

## C
```C
#include <stdio.h>

int main()
{
  printf("HelloWorld!\n");
  return 0;
}
```

## C++
```C++
#include <iostream>

using namespace std;

int main()
{
  std::cout << "HelloWorld" << std::endl;
  return 0;
}
```

## go
```go
package main

import "fmt"

func main() {
  fmt.Println("HelloWorld");
}
```
